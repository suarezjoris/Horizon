Cheatsheet & AI Technologies

🤖 AI Stack

  • LLM (Chat): Ollama local instance (Dolphin-Mixtral / Qwen / etc.)
  • Image Gen: ComfyUI via API (Stable Diffusion / SDXL)
  • Video Gen: ComfyUI (Image-to-Video models like SVD)
  • Second Brain: Local Markdown Zettelkasten + Ollama semantic extraction
  • RAG & Memory: ChromaDB + Nomic-Embed-Text + Native D3.js Neural Field
  • Web Search: DuckDuckGo (duckduckgo-search) + html2text

💬 Chat Commands

  • /clear : Clear the current chat history.
  • /docx <topic> : Create a professional Word document.
  • /ppt <topic> : Create a Claymorphic PowerPoint presentation.
  • /xlsx <topic> : Create an Excel spreadsheet.
  • /search <query> : Manually search the Second Brain vault.
  • /reindex : Rebuild the RAG vector index (ChromaDB).
  • /wiki : Fetch Wikipedia articles for your vault notes and save them as cross-linked knowledge files.
  • /save <note> : Save the last AI response directly to a vault note (e.g. /save music).
  • /consolidate : Ask the AI to refactor and link your Second Brain notes.
  • /file <path> : Load any file (Code, PDF, Docx, Pptx, Xlsx) into context.
  • /paste : Paste an image from the clipboard (or press F2).

⚡ Auto-Triggers (LLM side)

Horizon automatically invokes these capabilities when it detects certain keywords in its own responses:

  • SEARCH_WEB -> Deep Research via DuckDuckGo.
  • GENERATE_IMAGE -> Image generation via ComfyUI.
  • GENERATE_VIDEO -> Video animation via Cinema engine.
  • GENERATE_DOCX / PPTX -> Professional document creation.
